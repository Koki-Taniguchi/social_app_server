# social_app_server

こちらの[アプリ](https://github.com/Koki-Taniguchi/social_app)で使用する Cloud Functions のテスト環境

- functions-framework-go

### 必要なもの
- docker
    - functions-framework-goをコンテナで用いるために使用
- buildpack
    - functions-framework-goをコンテナで用いるために使用
- ngrok
    - スマホから叩けるようにするために使用する
- direnv
    - 環境変数さえ設定できればなんでもいい
